# 💜 CentangIn

> "Satu Centang, Selangkah Lebih Dekat dengan Impianmu."

**CentangIn** adalah aplikasi manajemen tugas (Todo List) sederhana namun elegan yang dirancang dengan fokus pada estetika modern. Proyek ini dibuat untuk mengimplementasikan manipulasi data **Array dalam JavaScript** serta teknik desain **Glassmorphism** yang profesional.

---

## ✨ Fitur Utama
- **Modern UI/UX**: Menggunakan tema *Deep Purple* dengan efek *Glassmorphism* (kaca transparan).
- **Responsive Design**: Tampilan optimal di berbagai perangkat, mulai dari desktop hingga smartphone.
- **Array Management**: Implementasi metode `push()`, `splice()`, dan `forEach()` untuk mengelola data tugas secara dinamis.
- **Interactive UI**: Animasi *smooth* saat menambah atau menghapus tugas menggunakan emoji (➕ & ➖).

## 🛠️ Teknologi yang Digunakan
- **HTML5**: Struktur semantik yang bersih.
- **CSS3**: Custom property, Flexbox, dan teknik `backdrop-filter` untuk efek blur kaca.
- **JavaScript (ES6)**: Manipulasi DOM dan logika Array.
- **Google Fonts**: Menggunakan font 'Montserrat' untuk kesan profesional.
